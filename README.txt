c.  Un archivo README en la raíz que describa al menos:
 
    Las fuentes para crear el proxy se encuentran en la raíz. Para crear el ejecutable, hay que pararse en la raíz y hacer:
        make all

    Las fuentes para crear el cliente se encuentra en la carpeta client. Para crear el ejecutable, hay que pararse en la raíz y hacer:
        make all
 
    El binario del cliente se encontrará en client y el del proxy en la raíz.

    Para ver las opciones, hacer "http -h" en la raíz o "httpdctl" en client para ver las opciones del proxy o el cliente respectivamente.

    Para ver los distintos comandos del cliente, ejecutar "help" en al correr el cliente. Recordar que éste espera comandos sin decir nada, y corta cuando le hacemor Ctrl+C.
